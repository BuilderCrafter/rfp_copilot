# Case Study: Municipal Service Request Modernization

## Client Context

A mid-sized municipality needed to replace email-based service request handling with a structured citizen request portal and internal case-management workflow. Departments included public works, permitting, waste management, parks, and citizen communications.

## Scope

Northstar delivered citizen request submission, status tracking, staff triage queues, role-based administration, email notifications, service category configuration, reporting dashboards, and integration with the municipality's document repository.

## Delivery

The project used discovery workshops, prototype review, iterative sprint delivery, integration testing, UAT with departmental users, staff training, and a phased go-live. Governance included weekly status meetings and monthly steering reviews.

## Outcomes

The municipality reduced manual email triage, improved visibility of request ownership, standardized department workflows, and created a reusable pattern for adding new service categories. The client highlighted source-traceable decision logs and UAT scripts as useful during acceptance.

## Relevant Notes

This case study supports public-sector experience, citizen portal implementation, workflow configuration, role-based administration, governance, training, and phased rollout claims. It does not include payment gateway implementation.
